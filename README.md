# Full-Stack Learning Notes

This repository is the home for my full-stack learning notes and internship preparation.

## Goal

Use this repo to:

- learn core full-stack concepts step by step
- write clear notes instead of scattered reminders
- track practice and mini projects
- prepare for internship interviews with focused review

## Folder Structure

### `00-roadmap/`

High-level study plans, weekly goals, and progress tracking.

### `01-foundations/`

Core web fundamentals such as HTML, CSS, JavaScript, TypeScript, browser basics, and Git.

### `02-frontend/`

Frontend topics such as React, state management, routing, UI patterns, performance, and accessibility.

### `03-backend/`

Backend topics such as Node.js, Express, APIs, authentication, and server architecture.

### `04-databases/`

Database notes covering SQL, PostgreSQL, schema design, indexing, and data modeling.

### `05-testing/`

Testing concepts including unit tests, integration tests, end-to-end testing, and debugging habits.

### `06-devops/`

Deployment, environments, CI/CD basics, Docker, cloud concepts, and production workflows.

### `07-system-design/`

Beginner-friendly system design notes for web apps, scaling, caching, queues, and architecture tradeoffs.

### `08-interview-prep/`

Interview review notes, behavioral stories, common questions, resume bullets, and final revision material.

### `09-leetcode/`

LeetCode practice notes, problem patterns, solution walkthroughs, and revision by topic.

### `10-projects/`

Mini projects, implementation notes, feature breakdowns, and lessons learned from building.

### `11-templates/`

Reusable note templates for topic notes, project writeups, and interview prep checklists.

### `12-resources/`

Curated learning resources, useful links, and reference material worth revisiting.

## How To Use This Repo

1. Start with `00-roadmap/` to define what to study next.
2. Write concept notes in the topic folders as you learn.
3. Practice coding problems in `09-leetcode/` to improve interview readiness.
4. Build small projects in `10-projects/` to turn theory into practice.
5. Review and summarize important material in `08-interview-prep/`.
